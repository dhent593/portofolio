# Tentang Saya

Lulusan Sistem Informasi dari Universitas Amikom Yogyakarta dengan ketertarikan mendalam pada pengembangan infrastruktur jaringan dan otomatisasi teknologi.

Saat ini saya berkarier di Els Service, dengan tanggung jawab penuh untuk memastikan stabilitas dan kelancaran operasional teknis dan administrasi. Fokus keahlian saya mencakup manajemen sparepart, administrasi keuangan, dan pengembangan sistem administrasi service.

Selain itu, saya aktif mengembangkan berbagai solusi berbasis low-code menggunakan Google Apps Script & AppSheet serta menggunakan bahasa pemrograman JavaScript untuk menunjang pembuatan sistem di PT.Senndyt Sarungtangan Kreatif. Solusi ini dirancang khusus untuk mengotomatiskan alur kerja bisnis internal seperti manajemen inventaris, sistem penggajian, dan sinkronisasi data operasional guna mengoptimalkan efisiensi tim secara signifikan.
